Страница error.jsp будет отображать информацию об исключении. 
Через глобальный объект pageContext в страницу передается контекст. 
Если при обработке запроса возникло какое-нибудь исключение, то метод 
pageContext.getException() возвратит это исключение в виде объекта Exception.


Элемент exception-type указывает, что обрабатываться будут исключения типа java.lang.Throwable. 
Элемент location определяет страницу, которая отправляется пользователю при возникновении исключении. В данном случае это error.jsp.
В итоге при обращении к сервлету, при выполнении которого возникла ошибка будет сгенерирована страница с исключением.
